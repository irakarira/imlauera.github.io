Como descargar una playlist entera de YouTube y unirla en un solo archivo

imla
19.03.2023
linux, command line tool
Índice
    • Descargar toda la playlist y guardarlos en archivos sin espacio para facilitar el trabajo usando ffmpeg
    • Guardar todo en un archivo mylist.txt para que ffmpeg pueda juntar los archivos

Descargar toda la playlist y guardarlos en archivos sin espacio para facilitar el trabajo usando ffmpeg

yt-dlp -f140 'https://youtu.be/playlist?list=PLff7Jl9WJY9tkiCxOEwAS67pFUBGQMtkE' -o '%(id)s.%(ext)s'

Guardar todo en un archivo mylist.txt para que ffmpeg pueda juntar los archivos

ls -l > mylist.txt agregar file '' a cada nombre de cada video

archivo del formato de mylist.txt a respetar:

# this is a comment
file '/path/to/file1.wav'
file '/path/to/file2.wav'
file '/path/to/file3.wav'

Lo juntamos con:

ffmpeg -f concat -safe 0 -i mylist.txt -c copy output.wav
